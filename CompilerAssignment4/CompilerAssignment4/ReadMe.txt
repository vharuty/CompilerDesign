HOW TO RUN

1. Copy source files into a directory, copy folder "files" into the same folder
 as  exe. 
2. Open the project in Visual Studio 2012 then run
3. In the begginning the program asks to enter file name => copy full path into the screen(for simplicity)
4. program outputs files into a ".\files\Output". The deep description what does output files contain is in the document.
